
KeenKeeper

A modern relationship management web app to help you stay connected with your friends and maintain meaningful relationships.

---

## 🚀 Live Link
👉 (https://keen-keeper1.netlify.app/)

---

##  Project Overview

KeenKeeper helps users track their interactions with friends and reminds them when to reconnect.  
It provides a clean dashboard, friend details, and a timeline system to log calls, texts, and video interactions.

---

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- Local Storage
- React Hot Toast

---

##  Key Features

Friend Management
- View all friends in a clean card layout
- Status-based color system (Overdue, Almost Due, On Track)
- Detailed friend profile page

Quick Check-In System
- Call, Text, Video buttons
- Automatically adds interaction to timeline
- Toast notification on action

Timeline Tracking
- View all interaction history
- Stores data using LocalStorage
- Filter by Call / Text / Video (if implemented)

Analytics (Stats Page)
- Pie chart showing interaction distribution
- Built using Recharts

### UI/UX
- Fully responsive (Mobile, Tablet, Desktop)
- Dark modern theme 
- Smooth hover effects and animations
